##　1.素数判定(`prime_check.py`)
　　素数判定のプログラムです。input()により入力された値の素数判定を行います。
  　
   このプログラムで使ったアルゴリズムの証明です。
　　命題　整数Nは、2以上かつ√N以下の整数で割ることができなければ、
　　　　　　　　　　　　√Nより大きく、かつN未満の整数でも割り切ることはできない。
　　※この証明では古典倫理をもとに考える
　　※ここで使う約数とは正の約数のことを指す
　　合成数の整数Nを素因数分解して現れる約数のうち、小さい順にP1,P2,P3,P4……とすると、P1**2<=Nなので、P1<=√N。
　　最も小さい約数pは2<=p<=√Nの範囲内にあるため、もし（2以上かつ√N以下の範囲内に約数がないのであれば、√Nより大きく、かつN未満の範囲にも約数は存在しない。）...①
　　〜①についての証明〜
　　①は（2≦p≦√Nとなる整数Nの約数pが存在しないなら、√N≦p'<Nとなる整数Nの約数p'は存在しない） ... ②と同義である。
　　∃1≦p≦√Nとなる整数Nの約数p ==> ∃N/pも整数Nの約数 …③
　　背理法を用いて、命題②の否定が真なら矛盾が発生することを示す。
 　②の否定：（2≦p≦√Nとなる整数Nの約数pは存在しないが、√N<p'<Nとなる約数Nの約数p'は存在する場合がある。）...④
　　実際に約数p'があると、③を用いて約数N/p’の存在を明らかにすることができます。この定義域は√N<p'<Nを変形すると2≦N/p’<√Nとなります。N/p'はNの約数なのに命題の2≦p≦√Nとなる約数pが存在しないことに反します。
　　よって、④は偽となるため①は真となります。
　　このため、元の命題も真となります。
